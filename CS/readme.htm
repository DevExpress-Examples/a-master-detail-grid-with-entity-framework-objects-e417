Developer Express Example
A master-detail grid with Entity Framework objects

Starting with version 2012 vol 1, it is possible to enable the master-detail feature for collection properties generated by Entity Framework. See the http://www.devexpress.com/scid=S136884 thread for details. This example demonstrates how to bind related entities to the XtraGrid and display them in master-detail views. See also: http://www.devexpress.com/scid=E2766

You can find sample updates and versions for different programming languages here:
http://www.devexpress.com/example=E417.
